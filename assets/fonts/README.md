# assets/fonts/

Optional: Place self-hosted font files here if you want to avoid
Google Fonts CDN (e.g., for offline use or performance).

The site currently loads all fonts from Google Fonts CDN:
- Cormorant Garamond (display headlines)
- Playfair Display (section titles)
- Poppins (navigation / UI text)
- Inter (body text)
- Dancing Script (handwriting / signature accents)

If you self-host, download WOFF2 files from Google Fonts and
update the @font-face declarations in css/style.css accordingly.
