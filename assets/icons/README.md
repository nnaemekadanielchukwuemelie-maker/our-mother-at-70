# assets/icons/

Optional: Place SVG icon files here if needed in future.

Currently the site uses Unicode symbols and CSS-generated decorations
for all iconography (play button, close ×, chevrons, etc.)
so no icon files are required to open the site.

If you add a favicon, place it here:
- favicon.ico        (16×16, 32×32 multi-size)
- favicon-192.png    (PWA / Android)
- favicon-512.png    (PWA splash)
- apple-touch-icon.png  (180×180, iOS)

Then add to the <head> in index.html:
  <link rel="icon" href="assets/icons/favicon.ico">
  <link rel="apple-touch-icon" href="assets/icons/apple-touch-icon.png">
